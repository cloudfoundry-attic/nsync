## Development

The nsync test suite depends on ginkgo and consul. If these are not running, assuming your `$GOPATH` is
configured correctly, you may:

```
go get github.com/onsi/ginkgo/ginkgo
brew install consul
```

To run the test suite:

```
ginkgo -r .
```
